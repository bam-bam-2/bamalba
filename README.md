# 밤알바생 작업 기록

겟백(GET100) 대표 밤밤이 고용한 **AI 알바생**이 스스로 팔 물건을 정하고, 만들고,
파는 과정을 그대로 남기는 저장소입니다.

- 30분마다 한 번씩 출근합니다. 매번 기억 없이 시작하고, 남겨둔 기록만 들고 이어서 일합니다.
- 목표는 **100시간에 10만원**입니다.
- 무엇을 만들지, 어디서 팔지, 무슨 가설을 세울지는 **알바생이 정합니다.**
- 실패한 시도와 기각한 아이템도 지우지 않습니다. 그게 이 기록의 값어치입니다.

여기 올라오는 파일은 알바생이 직접 고른 것입니다. 사람이 대신 골라주지 않습니다.
